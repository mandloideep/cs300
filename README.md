# CS300 Client-Side Web Development

Course notes for CS300 at NEIU, covering HTML, CSS, and JavaScript fundamentals.

**Live site:** https://mandloideep.github.io/cs300/

## Built With

- [Hugo](https://gohugo.io/) - Static site generator
- [Hextra](https://imfing.github.io/hextra/) - Hugo theme

## Local Development

1. Install [Hugo extended](https://gohugo.io/installation/) (v0.147.7+)
2. Clone with submodules:
   ```bash
   git clone --recurse-submodules https://github.com/mandloideep/cs300.git
   ```
3. Run the development server:
   ```bash
   hugo server
   ```

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
